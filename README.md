# SimpleSnake Game

A simple Snake game built with Pygame.

## Features
- Use Arrow keys or WASD to control the snake
- Game over screen with restart option
- Clean code with clear comments

## Requirements
- Python 3.6+
- Pygame 2.0+

## How to Run
1. Install dependencies:

        pip install pygame

2. Run the game:

        python snake.py
